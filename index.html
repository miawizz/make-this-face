Make This Face

Install this app
  • iPhone/iPad: Share → Add to Home Screen
  • Android: Menu → Add to Home Screen
  • Desktop (Chrome): Menu → Install “Make This Face”

Tap the button to get a prompt and make your face match it. Challenge: freeze your expression like a snapshot—no movement and no sound. Hold that face!
